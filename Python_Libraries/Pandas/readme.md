# Student Performance Analyzer

Analyzed real-world dataset of 2,000 students from Kaggle.

## What I Did
- Data cleaning and null value handling
- Exploratory Data Analysis (EDA)
- Correlation between study hours, parental education and scores
- Visualizations — heatmaps, boxplots, distribution charts

## Tools
Python, Pandas, NumPy, Matplotlib, Seaborn

## Dataset
Kaggle — Student Performance Dataset (2000 rows)
